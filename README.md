# LA Food Help

This is an application intended to help the food insecure in the Los Angeles area find aid and resources near them. 

## Food Dropoff Points
Users can browse a collated set of food dropoff points, filtered and sorted by their distance from the user. The site stores this data in a Firebase database.

## Restaraunt Donations
Restaraunts can report contributed leftover food to dropoff points, with the contributed food stored in the firebase database and collated and displayed to users.
